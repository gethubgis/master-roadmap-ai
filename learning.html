🗂
Obsidian
لبناء قاعدة معرفة شخصية
نظام ملاحظات مبني على Markdown يعيش على جهازك. فيه تربط الأفكار ببعضها مثل شبكة عصبية. كلما تعلمت شيئًا — دوّنه هنا بكلماتك الخاصة.
🕐 متى تستخدمه: بعد كل جلسة تعلم — سجّل ملخصًا بكلماتك في 3 دقائق
obsidian.md — مجاني تمامًا
📋
Notion
لتتبع الخطة ولوحة المتابعة
الأنسب لمتابعة تقدمك في الخطة، حفظ الروابط والمصادر، وإنشاء لوحة الـ Dashboard الأسبوعية. أقل عمقًا من Obsidian لكن أسهل للجداول والمتابعة.
🕐 متى تستخدمه: لتحديث لوحة المتابعة الأسبوعية وحفظ خطة الأسبوع القادم
notion.so — مجاني للاستخدام الفردي
🎴
Anki
لحفظ المعلومات والأوامر البرمجية
أداة Spaced Repetition تعتمد على علم النسيان. كل مفهوم أو أمر برمجي تتعلمه — أضفه كـ Flashcard. Anki سيعيد تذكيرك به في اللحظة التي أنت على وشك نسيانه.
🕐 متى تستخدمه: 10 دقائق كل صباح قبل جلسة الدراسة — مراجعة الكروت القديمة
apps.ankiweb.net — مجاني كامل
🐙
GitHub
Portfolio + Code Storage + Collaboration
ليس فقط لحفظ الكود — هو Portfolio تقني حي. كل مشروع مع README جيد هو إعلان عن مهاراتك. Activity graph الأخضر هو أقوى CV تقني.
🕐 متى تستخدمه: يوميًا — كل كود تكتبه يذهب لـ GitHub. اجعل Activity graph أخضر باستمرار.
github.com — مجاني للـ Public repos
⏱
Toggl Track
تتبع وقت الدراسة الفعلي
تتبع كم ساعة درست فعليًا كل يوم وعلى أي موضوع. الأشخاص يبالغون في تقدير وقتهم — هذه الأداة تُريك الحقيقة. الوعي بالوقت يُحسّن الإنتاجية.
🕐 متى تستخدمه: شغّله عند بداية كل جلسة دراسة، أوقفه عند النهاية — لا تخترعها
toggl.com/track — مجاني
🖥
VS Code
بيئة التطوير الاحترافية
المحرر الأفضل لـ Python وكل التقنيات في هذا المسار. ثبّت هذه الإضافات: Python Extension, Pylance, GitLens, GitHub Copilot (مجاني للطلاب), REST Client.
🕐 متى تستخدمه: كل وقت تكتب فيه كود — لا تستخدم أي محرر آخر لكي تتعود
code.visualstudio.com — مجاني كامل
💡 ترتيب الأولويات: ابدأ بـ VS Code يوم ١، ثم GitHub يوم ٣، ثم Obsidian للملاحظات من الأسبوع الأول. Anki من الشهر الثاني. Toggl إذا حسست أنك تضيع وقتك.
